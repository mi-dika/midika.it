# MiDika

MiDika website. Minimal, fast, and maintainable.

## Development

```bash
# Install
pnpm install

# Run
pnpm dev

# Check quality
pnpm check
```

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
